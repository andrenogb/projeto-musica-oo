# 🎵 Gerenciador de Avaliação de Músicas

Este projeto é um exercício prático de Java focado em **Programação Orientada a Objetos (POO)**. O objetivo foi criar uma estrutura que represente uma música, permitindo armazenar informações técnicas e calcular a média de avaliações dos usuários.

## 🚀 Tecnologias
* **Java 17** (ou a versão que você usou)
* **IntelliJ IDEA** (ou sua IDE de preferência)

## 🧠 Conceitos Aplicados
Durante o desenvolvimento, pratiquei:
- **Criação de Classes:** Estruturação da classe `Musica`.
- **Atributos:** Definição de estados como título, artista e ano de lançamento.
- **Métodos com e sem retorno:** 
    - `avalia(double nota)`: Para acumular valores e incrementar o contador.
    - `pegaMedia()`: Para realizar cálculos lógicos baseados nos atributos.
- **Instanciação de Objetos:** Manipulação da classe através da classe `Main`.

## 🛠️ Como rodar
1. Clone o repositório.
2. Certifique-se de ter o JDK instalado.
3. Execute a classe `Main.java`.
